# UI Showcase

Design system reference page displaying all shared components, color palette, typography, and interaction patterns.

## Sections

- **Component Library** — Tabs, SearchInput, Collapsible, Loading states
- **Color Palette** — Background, text, accent, and border CSS variables
- **Typography** — Headings, body text, monospace, font weights
- **Buttons** — All variants (primary, accent, danger), sizes, states
- **v4.3 Components** — PageHeader, Card, Toolbar, DataChip, InteractiveChip, InfoRow, MiniChart, compact Tabs
- **Cards** — Card grid with icons and descriptions
- **Data Table** — Sortable table with pagination
- **Toasts** — Success, error, warning, info variants
- **Modals & Overlays** — Modal dialog and tooltip positions
- **Empty States** — EmptyState component with action buttons
- **Interactions** — Counter, progress bar, loading variants, badge variants, collapsible sections
- **Animations** — Card stagger entrance, hover effects (lift, glow, scale)
- **Interactive Terminal** — Simulated terminal with command history
