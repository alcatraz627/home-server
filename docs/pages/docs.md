# Documentation

Browse all project documentation organized by category.

## Features

- **Categorized browsing** — Architecture, Reference, Setup, and Page Guides
- **Inline expansion** — Click any doc to expand and read inline
- **Search** — Filter documents by keyword
- **Hash navigation** — Link directly to a doc section via URL hash (e.g., `/docs#roadmap`)
- **Markdown rendering** — Full GitHub-flavored markdown with syntax highlighting

## Categories

| Category     | Description                                                                 |
| ------------ | --------------------------------------------------------------------------- |
| Architecture | System design, extending, widgets, keeper, roadmap, linux support, security |
| Reference    | API reference, deployment guide, auth implementation                        |
| Setup        | Setup guide, README                                                         |
| Page Guides  | Per-page usage documentation for all 30+ pages                              |
